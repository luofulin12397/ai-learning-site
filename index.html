训练管线

一次完整的深度学习训练，从原始数据到收敛模型，由八个紧密咬合的环节构成——以下按训练管线的时间顺序展开。

①

文本预处理与分词（Tokenization）

BPE 子词切分，构建 token↔ID 双向映射，定义模型词汇空间

②

Token → 向量：嵌入层（Embedding）

wte 稠密向量 + wpe 位置编码，离散符号进入连续可微空间

③

模型架构：Transformer 前向计算图

Masked Self-Attention + FFN + 残差 + LayerNorm，L 层堆叠

④

前向传播：从 Embedding 到 Logits

逐层 attention→FFN→归一化，lm_head 投影 + Softmax 出概率

⑤

损失计算：Cross-Entropy & 策略目标

CE Loss / J(θ) / DPO pairwise loss 定义优化方向

⑥

反向传播：链式法则与自动微分

∂L/∂z = p−y → Chain Rule 逐层回传 → PyTorch autograd

⑦

梯度优化：SGD → Adam → Trust Region PPO

θ←θ−α·∇J，PPO clip ≈ KL 约束，Adam 自适应学习率

⑧

训练循环与多阶段对齐

Rollout→GAE→Update 迭代收敛，SFT→RM→PPO 三阶段管线

∑

公式与概念详解

交叉熵损失、Attention 矩阵乘法、REINFORCE 梯度估计、PPO 替代目标、DDPM 逆向过程… 所有关键公式的完整推导，集中在一页。

知识要点词典

按四大分支整理的核心术语速查——每个术语附带一句话定义和对应章节。

大语言模型 · LLM

  • BPE字节对编码，按子词频率合并构建词表（Ch1）
  • Embeddingwte 矩阵将 token ID 映射为 dmodel 维稠密向量（Ch2–3）
  • Self-AttentionQ·KT·V 计算序列内 token 间依赖权重（Ch3）
  • Multi-Head多组独立的 Q/K/V 投影，并行捕获不同子空间（Ch3）
  • FFN两层全连接 + GELU，提供非线性变换能力（Ch3）
  • LayerNorm层归一化，沿特征维度标准化以稳定深层训练（Ch3）
  • Cross-Entropy−Σ y·log(p)，衡量预测分布与真实分布的差异（Ch2）
  • Softmax将 logits 指数归一化为概率分布（Ch2）
  • ALiBi / RoPE线性偏置 / 旋转位置编码，替代绝对位置编码（Ch3）
  • Autoregressive自回归生成：当前输出作为下一步输入，逐 token 预测（Ch2–3）

强化学习 · RL

  • MDP马尔可夫决策过程：(S, A, P, R, γ) 五元组形式化框架（Ch4）
  • Bellman Eq.V(s) = maxa[R + γ·Σ P·V(s′)]，价值递归定义（Ch4）
  • Q-Learning基于 Q 表的时序差分学习，off-policy（Ch4）
  • DQN神经网络拟合 Q 值 + Experience Replay + Target Network（Ch4）
  • Policy Gradient∇J = E[∇logπ · R]，直接优化策略参数（Ch5）
  • Actor-CriticActor 做决策 + Critic 估计价值，组合 PG 与 TD（Ch5）
  • GAE广义优势估计，λ 参数在偏差与方差间折中（Ch5）
  • PPO-Clipclip(πnew/πold, 1−ε, 1+ε)，Trust Region 的工程化实现（Ch6–8）
  • GRPO组内相对优势替代 Critic，省去价值网络（Ch9）
  • Trust Region信赖域约束，限制 KL(πold‖πnew) ≤ δ 防策略崩溃（Ch8）

RLHF 对齐训练

  • SFT监督微调，在人类标注的 (prompt, answer) 对上训练（Ch10）
  • RM奖励模型，对两个回答输出标量评分，学人类偏好排序（Ch10）
  • PPO-ptxPPO + 预训练梯度项 β·∇logP(x)，防止遗忘（Ch12）
  • KL Penalty−β·KL(πref‖πθ)，约束策略不偏离初始模型（Ch12）
  • DPO直接偏好优化，Bradley-Terry 模型下等价 RLHF 但只需分类 loss（Ch11）
  • Bradley-TerryP(y≻y′)=σ(r−r′)，偏好比较的经典概率模型（Ch11）
  • Rejection Samp.拒绝采样：RM 打分后保留高分样本做 SFT（Ch10）
  • GRPO-LLMGRPO 用于对齐：组内比较替代 reward 估计（Ch13）
  • DeepSeek-R1基于 GRPO 的推理增强模型，省去 RM 打分阶段（Ch13）
  • InstructGPTSFT → RM → PPO 三阶段对齐管线（Ch12）

多模态 AI

  • ViTVision Transformer，图像切 patch + 线性投影→Transformer（Ch14）
  • Patch Embed.图像块展平后经线性层投影为 token 向量（Ch14）
  • CLIP双塔对比学习，图文编码器共享嵌入空间（Ch15）
  • Contrastive对比损失：拉近匹配对、推远非匹配对的余弦距离（Ch15）
  • ClipCapCLIP 视觉特征 → 前缀 token → GPT-2 解码文本（Ch16）
  • DDPM前向加噪 xt=√ᾱtx₀+√(1−ᾱt)ε，反向 U-Net 预测噪声（Ch17–18）
  • CFG无分类器引导：ε̂ = εθ(xt) + w·(εc−εu)，控制文图一致性（Ch19）
  • U-Net编码-解码+跳跃连接的噪声预测骨干网络（Ch18）
  • DALL·E 2CLIP Prior 映射文本→图像嵌入 + 扩散 Decoder 生成（Ch20）
  • Stable Diff.潜空间扩散（VAE+UNet+CLIP），大幅降低计算开销（Ch19）
