# AI知识学习网站

基于左元《大语言模型、强化学习和多模态教程》（407 页，21 章）构建的交互式知识学习网站。

## 内容

- **大语言模型**：microgpt → LLM 原理 → GPT-2 实现
- **强化学习**：RL 基础 → 策略梯度 → PPO → GRPO
- **RLHF 对齐训练**：训练流程 → DPO → PPO 微调 → GRPO 复刻 DeepSeek-R1
- **多模态 AI**：Vision Transformer → CLIP → ClipCap → 扩散模型 → DALL-E2

## 特色

- 四大知识主线 + 分支线 + 交叉知识关联
- 16 个核心公式的详细推导（可点击跳转）
- 9 个机器学习类比框，用监督学习概念辅助理解强化学习
- 响应式设计，支持移动端

## 技术栈

- 纯 HTML/CSS，无框架依赖
- Space Grotesk + DM Sans + JetBrains Mono 字体
- 4 色分支编码：LLM=indigo / RL=emerald / RLHF=amber / 多模态=red
